familyFinance.NET
-----------------

http://www.stefanobianchini.net

E' un semplice programma di prova, sviluppato in .NET con DBMS Sqlite.
Sono presenti due cartelle:

- software, contenente il codice compilato del programma
- sorgenti, contenente il codice sorgente del programma, le librerie utilizzate, le icone e le immagini

Le immagini e le icone sono state scaricate da http://www.iconfinder.com
